# Public Assets Folder

## Save Your Logo Here

**IMPORTANT**: Save your ChatGenie logo as `logo.png` in this folder.

### Required File

```
public/
└── logo.png  ← Save your logo here (512x512px or 1024x1024px PNG)
```

### How to Save

1. Right-click your logo image
2. Save as `logo.png`
3. Place in this `public` folder

### Logo Specifications

- **Format**: PNG with transparency
- **Size**: 512x512px or 1024x1024px (square)
- **Name**: Exactly `logo.png` (lowercase)

### Optional Additional Sizes

For better browser support, you can also add:

- `favicon.ico` - 32x32px ICO format
- `favicon-16x16.png` - 16x16px
- `favicon-32x32.png` - 32x32px  
- `apple-touch-icon.png` - 180x180px

### After Saving

1. Refresh your browser (Ctrl + Shift + R)
2. Logo will appear in:
   - Sidebar header
   - Empty chat state
   - Browser tab (favicon)

### Troubleshooting

If logo doesn't appear:
- Check file name is exactly `logo.png`
- Check file is in `public` folder
- Hard refresh browser
- Check browser console for errors
